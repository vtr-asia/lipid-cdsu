# lipid-cdsu
台灣血脂 CDS 決策支援系統是一款基於 SMART on FHIR 標準開發的臨床輔助工具，專為台灣醫療情境設計，遵循「2025 台灣血脂管理專家共識建議」提供即時 LDL-C 達標評估與治療決策支援。 系統透過 FHIR 伺服器安全讀取病患資料與最新 LDL-C 值（LOINC 13457-7），根據風險分層（一般風險、高風險、非常高風險、極高風險）自動對應目標值（&lt;130、&lt;100、&lt;70 mg/dL），並以紅黃綠色視覺化決策區塊呈現控制狀態（良好／尚未達標／嚴重超標），同時給出具體建議：維持現有治療、提升他汀劑量、合併 Ezetimibe 或考慮 PCSK9 抑制劑等。
